# 个人简介网站 - 移动应用设计作业

## 项目简介
本项目为移动应用设计课程作业，是一个个人简介主题的展示网站。
网站采用现代化设计风格，包含个人信息、喜欢的美食、喜欢的电影三大板块，
以及详细的技能介绍子页面。

## 技术栈
- **HTML5** - 页面结构
- **CSS3** - 样式设计（响应式布局、CSS变量、动画效果）
- **JavaScript (ES6+)** - 交互逻辑
- **无框架依赖** - 纯原生实现，轻量高效

## 项目结构
```
移动应用设计作业_个人网站/
├── index.html          # 首页（三大板块）
├── more.html           # 了解更多子页面（技能介绍）
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── main.js         # 交互脚本
├── images/             # 图片资源目录

```

## 页面结构

### 首页 (index.html)
1. **顶部导航栏** - 固定导航，支持滚动高亮、移动端汉堡菜单
2. **Hero区域** - 个人介绍、技术标签、代码风格展示卡片
3. **个人信息板块** - 基本信息卡片、性格标签、兴趣爱好
4. **喜欢的美食板块** - 6种美食卡片，带渐变色图片和评分
5. **喜欢的电影板块** - 6部电影卡片，带海报风格背景和评分
6. **页脚Footer** - 快速链接、技术栈、联系方式、版权信息

### 了解更多页 (more.html)
- 技能掌握情况，带进度条动画
- 编程语言、专业技能、工具框架、软技能四大类
- 学习计划展示

## GitHub Pages 部署步骤

### 方法一：通过网页端部署
1. 在 GitHub 上创建新仓库
2. 将项目文件上传到仓库
3. 进入仓库 Settings → Pages
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main` / `root`
6. 点击 Save，等待几分钟后即可访问
7. 访问地址：`https://<你的用户名>.github.io/<仓库名>/`

### 方法二：通过Git命令部署
```bash
# 1. 初始化仓库
git init
git add .
git commit -m "初始提交：个人简介网站"

# 2. 关联远程仓库
git remote add origin https://github.com/<你的用户名>/<仓库名>.git
git branch -M main
git push -u origin main

# 3. 在设置中开启GitHub Pages
```
